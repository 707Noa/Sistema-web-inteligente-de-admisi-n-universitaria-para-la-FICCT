# CUP-FICCT---Sistema-Web-de-Admisi-n-Universitaria
CUP-FICCT - Sistema Web de Admisión Universitaria CUP-FICCT es una aplicación web desarrollada para administrar el proceso de inscripción, organización, evaluación y admisión de postulantes al curso preuniversitario de la Facultad de Ingeniería de Ciencias de la Computación y Telecomunicaciones.
